# 마크 다운
## 줄바꿈
스페이스 두번 치면 줄바꿈.
## 문단
한 줄의 공백을 준다.  
```
1번째 문단
1번째 문단

2번째 문단
2번째 문단
```

## 헤더
\# 하나 마다 h1에서 h6까지.  
h1이 가장 큰 헤더
```
# h1  
###### h6
```
## 강조
강조할 문자 앞뒤로 * 두개씩.  
```
**강조**
```
## 기울이기
기울게할 문자 앞뒤로 * 한개씩.
```
*기울기*
```
## 취소선
취소할 문자 앞뒤로 ~ 두개씩.
```
~~취소선~~
```
## 밑줄
```
<u>밑줄</u>
```
## 코드블록
\`\`\`언어 이름(소문자)  
코드 작성 부분  
\`\`\`
## 링크
```
<링크주소>
```
## 인용문
```
> 인용문
>> 중첩 인용문
```
## 구분선
```
***
---
```
